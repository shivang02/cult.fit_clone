## How To Use

Clone this repository and open index.html<br>
From your command line:

```bash
# Clone this repository
$ git clone https://github.com/shivang02/cult.fit_clone

# Go into the repository
$ cd cult.fit_clone

# Run the app
$ open index.html 
```
